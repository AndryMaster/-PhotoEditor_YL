Приложение фоторедактор "Abobe Photoshop":

Это приложение редактирует фотографии и может
менять палитру цвета, накладывать фильтры на изображение,
выпольнять точную обрезку изображения, а так-же его поворот.

Файл можно сохранять в разном типе "jpg" "jpeg" "png" и "bmp".
В приложении присутствуют 3 окна для выбораб обрезки и редактирования.
